# PostgreSQL Mastery Guide & Reference Repository

A comprehensive, categorized reference repository containing structured notes, commands, and advanced queries for PostgreSQL. This repository is designed to serve as a complete revision guide and a practical blueprint for integrating relational database architecture into modern backend workflows.

## Repository Overview

This project transitions from foundational database design to advanced procedural logic and performance optimization. Each file is fully executable and includes heavily commented, practical examples using a consistent schema.

## File Structure & Contents

| File | Core Subject | Concepts Covered |
|------|--------------|------------------|
| `01_Basics.sql` | Database Architecture | `CREATE`, `ALTER`, `DROP`, `TRUNCATE`, Data Types, Constraints |
| `02_CRUD.sql` | Data Manipulation | `INSERT`, `SELECT`, `UPDATE`, `DELETE`, `RETURNING`, Filtering, Sorting |
| `03_joins.sql`| Relational Mapping | Foreign Keys, `INNER`, `LEFT`, `RIGHT`, `FULL OUTER` Joins |
| `04_AggFunctions.sql` | Data Summarization | `GROUP BY`, `HAVING`, `COUNT()`, `SUM()`, `AVG()`, `MIN/MAX()` |
| `05_Advanced_queries.sql`| Complex Retrieval | Subqueries, CTEs (`WITH`), Window Functions (`RANK`, `PARTITION`) |
| `06_Performance.sql`| Database Optimization | Indexes (`B-Tree`), Views, `EXPLAIN ANALYZE`, Triggers |
| `07_Transactions.sql` | Data Integrity & NoSQL | `BEGIN`, `COMMIT`, `ROLLBACK`, `JSONB` querying and manipulation |
| `08_Functions_Procedures.sql`| PL/pgSQL Programming | User-Defined Functions, Stored Procedures, `IF/ELSE` Logic |
| `09_Recap.sql` | Quick Reference | Master syntax dictionary for rapid retrieval of all commands |
